# Barbearia Hilzer - Desafio do Barbeiro Dorminhoco

Este projeto é uma simulação clássica de um sistema concorrente, desenvolvida para demonstrar conceitos avançados de Programação Paralela e Sincronização de Threads em Java.




## Tecnologias

**Client:** Java 17+
## Como Rodar

Clone o repositório e acessar o mesmo:

```bash
  git clone git@github.com:douglas-franca/barbearia.git
  cd barbearia
```

Caso voce **NAO** tenha um IDE, como por exemplo o VSCode que ja compila e roda, pode serguir os seguintes comandos:

```bash
    javac -d bin src/barbearia/*.java
````

E depois:

```bash
    java -cp bin barbearia.Main
```
